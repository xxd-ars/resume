# Job Search 2025

Repository for 2025 job application materials, focusing on Data Science, Quantitative Finance, and Engineering roles.

## Directory Overview

*   **`cv/`**
    Core LaTeX resume files (`resume.tex`, `resume_fr.tex`) and generated PDFs.
*   **`quant/`**
    Study materials and code specifically for Quantitative Finance preparation.
*   **`project_sum/`**
    Detailed summaries of key projects (STAR method) for interview discussions.
*   **`interview/`**
    General interview prep, including algorithms, system design, and behavioral questions.
*   **`admin_docs/`**
    Official documents: transcripts, diplomas, visa status, and internship agreements.
*   **`archives/`**
    Legacy files and materials from previous searches (e.g., 2024 internships).
*   **`templates/`**
    Base LaTeX templates for resumes or cover letters.

## Quick Start

**Compile Resume:**
Navigate to the `cv` directory and compile using `xelatex` (recommended for font compatibility):

```bash
cd cv
xelatex resume.tex      # Compile English Resume
xelatex resume_fr.tex   # Compile French Resume
```
